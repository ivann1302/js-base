# JS Base

Проект представляет собой базу знаний в сфере frontend-разработки и смежных областях.

## Функциональность

- **База знаний** - структурированная информация по категориям (HTML, CSS, JavaScript, TypeScript, React, Redux, Next.js, Networks)
- **База задач** - задачи разной сложности (легкая, средняя, тяжелая) с возможностью отслеживания прогресса
- **Полезные материалы** - коллекция ссылок на внешние ресурсы
- **Поиск** - полнотекстовый поиск по базе знаний, задачам и ресурсам с подсветкой совпадений
- **Персистентность данных** - сохранение статуса выполнения задач в localStorage, состояния раскрытых категорий в sessionStorage
- **Адаптивный дизайн** - поддержка мобильных и десктопных устройств

## Структура проекта

Проект организован по методологии Feature-Sliced Design:

- **app** - инициализация приложения, провайдеры, роутинг
- **pages** - страницы приложения (главная, задачи, база знаний, ресурсы, результаты поиска)
- **widgets** - крупные UI компоненты (header, footer, aside, breadcrumbs, iconGallery)
- **features** - функциональные возможности (поиск с выпадающим списком и страницей результатов)
- **entities** - бизнес-сущности (задачи, знания, ресурсы)
- **shared** - переиспользуемые компоненты и утилиты (кнопки, иконки, хуки, утилиты)

## Команды

- `npm run dev` - запуск dev сервера
- `npm run build` - сборка проекта
- `npm run check` - полная проверка (формат, линт, типы)
- `npm run check:fix` - автоматическое исправление всех проблем
- `npm run commit` - создание коммита через Commitizen
- `npm run docker:dev` - запуск в Docker для разработки
